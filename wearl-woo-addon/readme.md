== Description ==
A modern boilerplate addon for WooCommerce built by Wearl Technologies.

== Installation ==
1. Upload `wearl-woo-addon` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
