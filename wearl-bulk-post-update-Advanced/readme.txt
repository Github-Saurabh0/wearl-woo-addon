Change the Post Update date for posts and pages in one click. This helps keep your site content looking fresh for search engines.

== Features ==
* Update Posts Modified date in bulk
* Update Pages Modified date in bulk
* Random distribution across a selected time window
* Custom date range support
* Per-category and per-page selection
* Option to update Published date, Modified date, or Both

== Installation ==
1. Upload the folder 'wearl-bulk-post-update-date' to /wp-content/plugins/
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to the admin menu item "Bulk Post Update Date".
